What is Comfortable Together

Allows you and your team to create your own personal comfort zones regarding some factors as illumination comfort, sound comfort, Temperature comfort and Humidity comfort. the Websites connects the location of all members and using the several standards codes an analysis is done to the measured data, then feedback is given whether the user is in the comfort zone or not. If the user is not in the comfort zone the websites give suggestions to the user to allow him with the standard limit of the comfortable zone, if the user is still not comfortable feedback is taken from the user and accordingly other suggestions are given to the user and this process is repeated until the user is comfortable and then the standard comfortable ranges are adjusted to his personal comfort. Finally, the measurement data of each of the team members are represented on the website in form of chart giving a final indication whether the overall group is comfortable or not.
